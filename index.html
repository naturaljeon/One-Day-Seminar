하루의 3단계

강의가 아니라, 결과물을 가져가는 하루

세 가지 결과물을 직접 만들어 노트북에 담아 갑니다. 앞에서 만든 것을 뒤에서 그대로 이어 씁니다.

01
Part 1 · 오전

옵시디언+클로드로 '나만의 영업 DB' 만들기

프롬프트·프로젝트·스킬·인스트럭션 세팅을 직접 다루면서, 고객명을 입력하면 가입 이력·갱신일·가족 구성이 한 번에 나오는 영업 데이터베이스를 구축합니다.

  • ✓보험상품 DB: 상품·보장·조건을 속성으로 정리
  • ✓고객 DB: 가입 이력, 갱신일, 가족 구성, 유형 태그
  • ✓엑셀·수첩에 흩어진 기존 데이터 옮겨오기
02
Part 2 · 오후 1부

상담 녹음 → 텍스트 변환 → 고객 DB 자동 등록

커넥터·MCP로 외부 파일을 연결합니다. 상담 녹음을 Whisper로 텍스트화하고, 핵심 정보를 뽑아 고객 DB에 자동으로 기록하는 파이프라인을 만듭니다.

  • ✓녹음 파일을 텍스트로 자동 전사
  • ✓상담 핵심 정보 추출·정규화
  • ✓올바른 고객 레코드에 자동 등록
03
Part 3 · 오후 2부

맞춤 제안서 자동 생성 + 유형별 메일·문자 발송

Claude Cowork와 Loop process(오케스트레이터-워커-QA)로, 고객을 선택하면 제안서가 인쇄 가능한 형태로 생성되고 유형별 고객에게 맞춤 메시지를 보낼 준비까지 완료합니다.

  • ✓고객 조건 → 상품 조합 제안서 자동 생성
  • ✓'갱신 임박', '자녀 보험 미가입' 등 유형별 추출
  • ✓유형별 맞춤 메일·문자 발송 준비까지

FAQ

자주 묻는 질문

컴퓨터를 잘 다루지 못하는데 따라갈 수 있나요?+

네. 엑셀로 고객 명단을 정리해본 정도면 충분합니다. 모든 과정을 화면 그대로 따라 하며, 막히면 현장에서 바로 도와드립니다.

옵시디언이 뭔가요? 유료인가요?+

옵시디언(Obsidian)은 무료 노트·데이터베이스 프로그램입니다. 모든 데이터가 내 컴퓨터에 파일로 저장되어, 고객 정보를 외부 서버에 올리지 않고 관리할 수 있습니다.

상담 녹음을 다뤄도 개인정보가 안전한가요?+

세미나 실습은 가상의 예시 녹음·데이터로 진행합니다. 등록은 본인 컴퓨터의 옵시디언에 저장되는 구조라, 실제 적용 시에도 개인정보를 외부에 맡기지 않습니다.

무엇을 준비해야 하나요?+

노트북 한 대면 됩니다(Windows/Mac 모두 가능). 평소 쓰시는 고객 명단 엑셀이 있다면 가져오세요. 마지막 상담 시간에 내 데이터 기준으로 적용을 도와드립니다.

하루 만에 정말 다 만들 수 있나요?+

하루 전체를 실습으로 구성했기 때문에 가능합니다. DB 구축(Part 1) → 녹음 자동 등록(Part 2) → 제안서·발송 자동화(Part 3) 순으로 이어지며, 앞 파트에서 만든 결과물을 다음 파트에서 그대로 재료로 씁니다. 단, 완성도는 당일 진행 속도와 본인 환경에 따라 달라질 수 있습니다.
